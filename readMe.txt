# Braha Petru Bogdan: track progress

	This project contains all my work in the object-oriented programming class: assignments, laboratory explanations, extra work and documentation. This setup helps me track, meticulously, my progress as well as my files. To review my development, it's important for me to have everything in one place.

# Structure of files - Windows' file system:
- homework folder = all assignments (e.g., H1: Homework + Lab 1 + Extra)
- other CPPs and headers outside of that folder = references, examples of syntax, processes

# Structure of my files - VS's management:
- filters for my homework - denoted "Laboratory <nr>"
* filters for code examples - they do not have anything to do with the actual class

# Design:
- oop_main.cpp is the principal CPP.
- an assignment (e.g., H3) is a folder (subfolder of "Homework") 
- in his location, there are multiple folder_child, each one represents an exercise. 
- the files of a task are placed in folder_child of the folder_assignment; 
* !no matter how many files an exercise has, there will be one more file in the folder_assignment: name_of_homework_main.cpp (this file introduces the main functions for every task of an assignment such that they can be called in the main function of oop_main.cpp)
